Place your hero image here and name it:

    hero_background.jpg

Example:

    assets/hero/hero_background.jpg

Supported formats:

    * JPG  (hero_background.jpg)
    * JPEG (hero_background.jpeg)
    * PNG  (hero_background.png)

Recommended image size:

    1920 x 1080 pixels

The app checks for these filenames in order:
  1. hero_background.jpg
  2. hero_background.jpeg
  3. hero_background.png

The first match found will be used as the hero background.
If no image is found, the app uses a dark gradient fallback automatically.

Contractors can replace this image with their own branding photo.
